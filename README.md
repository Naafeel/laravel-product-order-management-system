# Laravel Product & Order Management System

This is my Month 1 Internship Project for Zulacart.

Built with Laravel 12 + MySQL.

## Features (Will be added weekly)
- Home Page
- Product Listing
- Product Details
- Cart System
- Order Management
- Admin Panel (basic)

## Setup
1. Clone the repo
2. Run `composer install`
3. Copy `.env.example` to `.env`
4. Run `php artisan serve`
