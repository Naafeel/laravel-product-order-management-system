# Zulacart Public User Flow Diagram

## Main Customer Flow

1. **Homepage** 
   → See Hero Banner + Featured Products
   → Click "Shop Now"

2. **Product Listing Page** 
   → Browse products
   → Filter / Search (future)
   → Click on product

3. **Product Details Page**
   → Read description, price, images
   → Click "Add to Cart"

4. **Cart Page**
   → Review items
   → Update quantity / Remove
   → Click "Checkout"

5. **Login / Register** (if not logged in)
   → Then proceed to Checkout

6. **Order Placement** (future)
